# 🏠 SuperKost — Kost & RooMatch Finder

SuperKost adalah aplikasi pencarian hunian berbasis web yang membantu pengguna menemukan kost, kontrakan, atau apartemen sesuai kebutuhan mereka.  
Selain itu, aplikasi ini juga menyediakan fitur **Roommate Matching** untuk membantu pengguna menemukan teman sekamar yang cocok berdasarkan preferensi dan gaya hidup.

---

## ✨ Fitur Utama

### 🔍 Pencarian Hunian
- Smart Filter (harga, lokasi, fasilitas, tipe hunian)
- Smart Sorting
- Wishlist / Favorit Hunian
- Real-time Map View
- Perbandingan Hunian

### 👥 Roommate Matching
- Roommate Survey
- Sistem rekomendasi roommate
- Roommate Request
- Chat antar pengguna

### 🏢 Manajemen Properti
- Tambah / edit / hapus hunian
- Monitoring status hunian
- Kelola penyewa

### ⭐ Sistem Pendukung
- Login & Register
- Review & Rating
- Notifikasi Sistem
- Dashboard Admin

---

## 👤 Role Pengguna

### 1. Pencari Hunian
Pengguna yang mencari kost, kontrakan, atau apartemen.

### 2. Pemilik Properti
Pengguna yang mengelola dan menyewakan hunian.

### 3. Admin
Pengelola sistem dan moderasi aplikasi.
